# 🩺 Advanced Risk Modeling for Medical Applications

A comprehensive collection of **graduate-level lecture slides**, **R code examples**, datasets, and practical tutorials on **survival analysis** and **modern risk modeling methods** for medical research.

The material is primarily based on *Modelling Survival Data in Medical Research (4th Edition)* by **David Collett** and is supplemented with practical examples in **R**.

---

# 📚 Modules

## 📖 Module 1 — The Basics (Chapters 1–2)

This module introduces the foundations of survival analysis and nonparametric methods for analyzing time-to-event data.

### Topics include:

- 📌 Introduction to Survival Analysis
- 🏥 Applications of Survival Analysis
- ⏳ Censoring Mechanisms
- 📈 Survival Function
- ⚠️ Hazard Function
- 📊 Cumulative Hazard Function
- 📉 Kaplan–Meier Estimator
- 📉 Nelson–Aalen Estimator
- 🎯 Confidence Intervals
- ⌛ Median Survival Time
- ⚖️ Log-Rank Test
- 📋 Wilcoxon Test
- 📋 Peto–Peto Test
- 📚 Stratified Log-Rank Tests

---

## 📖 Module 2 — The Cox Regression Model (Chapters 3–4)

This module focuses on semiparametric regression models for assessing the relationship between covariates and survival outcomes.

### Topics include:

- 📈 Cox Proportional Hazards Model
- 🧮 Partial Likelihood Estimation
- 📊 Hazard Ratios
- 🛠️ Model Building
- 🎯 Variable Selection
- 🌿 LASSO for Cox Regression
- 📈 Nonlinear Effects
- 📐 Fractional Polynomials
- 👥 Risk-Adjusted Survival Curves
- ✅ Model Validation
- 🎯 Concordance Index (C-index)
- 📉 Time-Dependent ROC Curves
- 🔍 Residual Diagnostics
- 📊 Cox–Snell Residuals
- 📊 Martingale Residuals
- 📊 Deviance Residuals
- 📊 Schoenfeld Residuals
- ⏱️ Testing the Proportional Hazards Assumption
- 🔬 Influential Observations

---

## 📖 Module 3 — Parametric Regression Models (Chapters 5–7)

This module introduces fully parametric approaches for modeling survival data and compares them with the Cox model.

### Topics include:

- 📈 Exponential Regression
- 📈 Weibull Regression
- 📈 Gompertz Regression
- 📈 Log-Normal Models
- 📈 Log-Logistic Models
- 📈 Generalized Gamma Models
- 🚀 Accelerated Failure Time (AFT) Models
- 🏆 Model Selection
- 📖 Interpretation of Parameters
- 🔍 Model Assessment
- 🎯 Survival Prediction
